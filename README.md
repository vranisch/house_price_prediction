# üè† House Price Prediction Web App

A Flask web application that predicts house prices using a machine learning model trained with scikit-learn.

## Features
- Web form interface for user input  
- Real-time house price prediction  
- ML pipeline with preprocessing (imputation, scaling, encoding)  
- Flask backend + HTML frontend  

---

## Dataset
This project uses the Housing Prices dataset from Kaggle:  
https://www.kaggle.com/datasets/yasserh/housing-prices-dataset/data

You need to download the dataset manually and place the file as: Housing.csv

in the project root directory.
